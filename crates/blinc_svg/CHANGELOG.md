# Changelog

All notable changes to `blinc_svg` will be documented in this file.

## [0.1.13] - 2026-02-18

### Added

#### CSS Animations for SVG Elements
- CSS animations for `fill`, `stroke`, `stroke-width`, `stroke-dasharray`, `stroke-dashoffset`
- SVG path morphing via `d` attribute animation in @keyframes
- Tag-name selectors (`path`, `circle`, `rect`, etc.) for targeting SVG sub-elements
- CSS transform inheritance for SVGs from parent divs

### Fixed
- SVG clipping corrections
